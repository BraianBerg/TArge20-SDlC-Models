ERD Description

Table PrimaryColor

  • The table holds Primary Colors and foreach Primary Color there is Primary key, The Colors are taken form the table and displayed on ui and based on what color is selected then proper Shades are displayed.
  • Primary Key is the unique identifier foreach color, PrimaryColorName is the color
  • This tables Primary key is taken form this table and put in to Shades table as an foreing key and is assigned to proper shade.

Table Shades

  • The table holds Shades, foreach Shade there is assigned proper Primary Color, in the ui when the Primary color is selected it will display all the Shdes assigned to the Primary Color
  • Primary key is the Shades unique identifier, Foreing key is the PrimadyColorId and each Shade has it, and ShadesName is the Shade it self
  • The Primary Key is assigned to every Shade and every shade allso has Foreing key form Primary Colors tabel, that makes bonds the Primary Color and shade together
